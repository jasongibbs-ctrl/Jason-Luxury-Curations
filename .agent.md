name: jason-luxury-agent
summary: "Agent for editing the Jason Luxury Curations website with inline in-page image embedding and lightweight HTML/CSS tweaks."
---

## Use when
- You want to modify `index.html` or other site pages in this repository.
- You need to embed images directly in HTML as base64 data URIs (no external image files).
- You want fast small tweaks: hero text, colors, layout, fonts, responsiveness.

## Do not use when
- You need full backend or server-side logic (Node, API, database).
- You need heavy asset pipeline changes (webpack, npm install, large video embeds).

## Preferred workflow
1. Open requested HTML file and inspect existing structure.
2. Apply minimal direct edits to match instructions (inlining images, text updates, sections).
3. Validate output by checking for valid HTML and proper data URI syntax.

## Example prompts
- "Embed jason.jpg into `index.html` as an inline base64 image URI."
- "Change the header style and add a new brand section in this page." 
- "Convert hero image to embedded SVG or JPEG data URI directly in HTML."
